# Aphelion-rs: A Modern Linux Package Manager

Aphelion-rs aims to revolutionize Linux package management by offering a modern, optimized, and user-friendly solution written in Rust.

## Project Overview

Aphelion-rs is designed to be a state-of-the-art Linux package manager with initial focus on Debian and Ubuntu distributions.

## Technical Stack

The project is built using Rust and leverages libraries like Tokio for concurrency, structopt for CLI parsing, and PostgreSQL for database operations.

## Development Environment

The project uses GitHub for version control, MIT as the license, and Docker for development and testing.

## Installation

### Prerequisites

- Rust and Cargo installed on the system
- Git for source code cloning (if installing from source)

### Installation Methods

1. **Via Cargo (Preferred)**

    ```bash
    cargo install aphelion-rs
    ```

2. **From Source**

    ```bash
    git clone https://github.com/tzervas/aphelion-rs.git
    cd aphelion-rs
    cargo build --release
    ```

3. **Pre-compiled Binaries**
    Pre-compiled binaries will be offered for various platforms as GitHub releases, available at <https://github.com/tzervas/aphelion-rs/releases>.

4. **Package Managers**
    `.deb` or `.rpm` packages will be provided for Linux users.

5. **Docker**
    A Docker image will be provided for containerized solutions.
